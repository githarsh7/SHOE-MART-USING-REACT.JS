
# ShoeMart — Premium Sneaker Store UI

A modern and premium sneaker e-commerce frontend built using **React.js** and **CSS3**.  
ShoeMart delivers a clean and stylish shopping experience with responsive layouts, reusable components, dynamic cart functionality, and elegant UI styling.

---

## Features

- Modern premium UI design
- Fully responsive layout
- Dynamic cart counter
- Premium sneaker showcase
- Smooth hover animations
- Custom gradient styling
- Reusable React components
- Clean and scalable code structure

---

## Tech Stack

| Technology | Usage |
|------------|-------|
| React.js | Frontend Framework |
| CSS3 | Styling & Animations |
| JavaScript (ES6+) | Application Logic |
| Google Fonts | Typography |

## Installation

### Clone the Repository

```bash
git clone https://github.com/githarsh7/SHOE-MART-USING-REACT.JS.git
```

### Navigate to the Project Folder

```bash
cd shoemart
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```
